# 🚀 Task Management Web Application

A dynamic and responsive Task Management Web App built using **HTML, CSS, and JavaScript**.

## ✨ Features
- ➕ Add Tasks
- 🗑 Delete Tasks
- ✅ Mark as Completed
- 💾 Data persistence using LocalStorage
- 🔄 Real-time DOM updates
- 📱 Responsive Design

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Browser LocalStorage

## 🧠 Concepts Used
- DOM Manipulation
- Event Handling
- Array Manipulation using splice()
- Input Validation using trim()
- JSON Parsing & Stringifying
- Conditional Rendering
